Modal with keyboard trap

This is a link

Make note that there are 3 elements within this modal that should naturally obtain keyboard focus. Normally tabbing would allow the user to tab through these elements and ultimately on elements outside of this modal. That is not what we want.

Here we see that tabbing will loop round the elements present in the modal. When the modal is closed, keyboard focus will go back to the elemnt that maintained focus prior to the modal opening and user can continue to tab through the document as normal
